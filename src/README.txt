# murder_mystery_update_v17

## 変更内容

- 「インデックス」を「捜査資料」に名称変更。
- 特別調査シートの調査画面上の専用枠を削除。
- 特別調査シートは捜査資料から開き、GMが任意の参加者へ送信可能。
- GMの捜査資料には全シート・全調査カード・全特別調査シートが入ります。
- 密談中、所有カードを密談相手にだけ見せるだけでなく、密談相手へ譲渡可能。
- 譲渡されたカードは所有者が譲渡先に変わり、左上マーカーも譲渡先の固有色に変化。
- 調査カード内の取得者名表示を削除。取得者は左上マーカーで判別します。
- 雷マークが0になると、未取得の調査カードを新たに取得できません。取得済み・閲覧権限ありのカードは引き続き閲覧できます。
- タイマー機能を削除。

## 注意

Firestoreの既存ルームには古いデータが残るため、反映後は新しいルームで確認してください。
