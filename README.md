# 名前だけが生きている

## 概要
5人用マーダーミステリーをWeb上で進行するためのゲーム支援ツールです。

キャラクター選択、調査カードの取得、フェーズ進行、推理、投票、エンディング分岐までを一連の流れとして実装しました。


## デモ
[デモページはこちら](https://murdermystery-f537e.web.app)

## GitHub
[GitHubリポジトリはこちら](https://github.com/lucky-muscle/murder-mystery-app)

- React
- TypeScript
- Firebase Hosting
- Git / GitHub

## 主な機能
- キャラクター選択
- 重複選択防止
- フェーズ進行管理
- 調査カード管理
- 調査マップ表示
- 推理・投票
- エンディング分岐
